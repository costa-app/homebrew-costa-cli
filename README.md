# Homebrew Tap for costa-cli

This repository contains the Homebrew formula for installing
[`costa-cli`](https://github.com/costa-app/costa-cli).

## Installation

First, add the tap:

```bash
brew tap costa-app/costa-cli
```

Then install the CLI:
```bash
brew install costa-cli
```

Or install directly:
```bash
brew install costa-app/costa-cli/costa-cli
```

## Updating

To upgrade to the latest version:
```bash
brew update
brew upgrade costa-cli
```

## Issues

Issues with the CLI itself should be filed in the main repository:
https://github.com/costa-app/costa-cli

Issues related to the Homebrew formula (installation, checksums, etc.)
can be filed in this repository.
